⚠️ CORS 주의사항:
  • localhost:3000에서 테스트할 때 브라우저 CORS 정책으로 인해 일부 요청이 차단될 수 있습니다.
  • Auth Server (15001)와 Sentinel (15004)은 직접 호출하므로 CORS 헤더가 설정되어야 합니다.
  • 다른 서비스들은 API Gateway (15000)를 통해 호출되므로 Gateway의 CORS 설정이 중요합니다.
  • 문제 발생 시 브라우저 개발자 도구 콘솔을 확인하세요.
ℹ️ 서비스 구조: Auth Server (15001) → 직접 호출 | Gateway (15000) → User/Team Service | Sentinel (15004) → 직접 호출
